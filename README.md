# 99Bottles

Elixir implementation of default [99 Bottles of OOP](https://sandimetz.com/99bottles-sample-ruby) exercise.

## What you need to run this

* Elixir @^1.4
* mix

## Running the tests

You can run all of them using the `test` or `test.watch` mix tasks:

```
$ mix test
$ mix test.watch
```
